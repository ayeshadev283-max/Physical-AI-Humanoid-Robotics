# Glossary

## Physical AI Fundamentals

**Embodied AI**: AI systems with physical bodies interacting through sensors/actuators.

**Physical AI**: AI operating in the physical world via perception, reasoning, actuation.

**Subsumption Architecture**: Layered reactive controllers (Brooks, 1991).

## Core Robotics

**DOF** (Degrees of Freedom): Independent parameters defining configuration.

**Forward/Inverse Kinematics**: End effector pose from joints or vice versa.

## ROS 2

**Node**: Executable process.

**Topic**: Publish-subscribe channel.

**TF**: Transform tree for coordinate frames.

**URDF**: XML robot description.

## Perception

**Sensor Fusion**: Combining sensors for accuracy.

**IMU**: Accelerometer + gyroscope.

**LiDAR**: Laser distance measurement.

## Simulation

**Digital Twin**: Virtual replica for testing.

**Sim-to-Real Gap**: Simulation vs reality performance difference.

## Isaac Ecosystem

**Isaac Sim**: High-fidelity simulator.

**Isaac Gym**: GPU-accelerated RL.

**Isaac ROS**: Hardware-accelerated ROS 2 packages.

## VLA Models

**VLA**: Vision-Language-Action models.

**OpenVLA**: 7B parameter model.

**RT-2**: Google DeepMind VLA.

## Control

**ZMP**: Zero Moment Point stability.

**MPC**: Model Predictive Control.
